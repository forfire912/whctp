# 快速开始指南

## Windows用户（5分钟快速启动）

### 前置条件
- ✅ Windows 10/11
- ✅ 已安装Python 3.7+
- ✅ 已安装MySQL数据库

### 快速步骤

1️⃣ **安装依赖**
```cmd
pip install pymysql
```

2️⃣ **配置连接**
- 复制 `config.json.example` 为 `config.json`
- 修改数据库密码和CTP账户信息

3️⃣ **启动程序**
```cmd
双击运行 run.bat
```

4️⃣ **连接系统**
- 填写配置信息
- 点击"连接"按钮
- 开始下载和查询数据

### 详细安装

请参考 [WINDOWS_INSTALL.md](WINDOWS_INSTALL.md) 获取完整的安装和配置指南。

## 目录结构

```
whctp/
├── main_gui.py              # 主界面程序 ⭐启动入口
```

## 注意事项

⚠️ **重要提醒**
- 配置文件包含敏感信息，请妥善保管
- CTP账户信息需从期货公司获取
- 建议在模拟环境先测试
- 定期备份数据库

⚠️ **系统要求**
- 仅支持Windows平台（包含Windows DLL）
- 需要稳定的网络连接
- 确保MySQL服务运行正常

⚠️ **交易时间**
- 夜盘：21:00 - 次日02:30
- 日盘：09:00 - 15:00
- 避免非交易时段连接
